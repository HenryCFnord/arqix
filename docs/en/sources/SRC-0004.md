## RFC 4180 CSV Format

The common format for comma-separated-values files (Shafranovich, 2005), captured as the norm behind this repository's CSV surfaces.
The trace matrices, the normative-statement export, and the publisher's CSV-to-table conversion quote RFC 4180 for their quoting rules: fields split on unquoted commas, quoted fields unwrapped, doubled quotes collapsed.
