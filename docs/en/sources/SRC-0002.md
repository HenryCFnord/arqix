## RFC 2119 Requirement Level Key Words

The BCP 14 key-word definitions (MUST, SHALL, SHOULD, MAY, and their negations), captured as the normative base of this repository's requirement language.
The requirements style guide `docs/en/processes/requirements-style-guide.md` restricts authors to the SHALL/SHOULD/MAY subset, and the requirements checker enforces that subset (REQ-01-01-11-06); this record pins the definition text those rules interpret.
