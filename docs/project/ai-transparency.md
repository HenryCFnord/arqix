# AI Transparency

AI is used deliberately in this project, but not as a substitute for responsibility.

## What AI Is Used For

- brainstorming design alternatives
- pressure-testing structure and naming
- drafting and revising documentation
- coding support for small implementation steps
- identifying gaps and edge cases worth reviewing

## What AI Is Not Allowed To Do By Itself

- make final architectural decisions without human judgment
- silently redefine project scope
- introduce code or text without review
- create the illusion of maturity or completeness

## Human Responsibility

Human review remains the control point for what enters the repository.

That includes:

- checking technical correctness
- removing vague or inflated claims
- aligning changes with the project's actual goals
- deciding which tradeoffs are acceptable

The standard here is straightforward: AI may accelerate thinking and drafting, but ownership stays human.
